# traffic-abm-parallel

Simulación Basada en Agentes (ABM) de tráfico urbano con motor secuencial y paralelo (paralelización por agentes).

## Requisitos

- Java 17+
- Maven 3.9+

## Compilar

```bash
mvn clean package
```

El JAR se genera como:

- `target/traffic-abm.jar`

## Ejecutar (placeholders iniciales)

```bash
java -jar target/traffic-abm.jar --help
```

Los comandos finales y parámetros exactos se consolidarán conforme se implemente el motor (según `specs/spec.md` y `plans/plan.md`).
